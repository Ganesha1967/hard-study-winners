Официальный веб-портал сообщества **Hard Study Winners** - пространства для дисциплины, продуктивности и совместного роста



## Технологический стек:

- **HTML5**
- **CSS3**
- **Tailwind CSS**
- **JavaScript**


## Локальный запуск

```bash
git clone https://github.com/Ganesha1967/hard-study-winners.git

cd hard-study-winners
```

После этого просто откройте `index.html` в браузере



## Контрибьютинг

Мы приветствуем любые улучшения проекта!

Если хотите внести изменения:

1. Сделайте **Fork** репозитория
2. Создайте новую ветку

```bash
git checkout -b feature/amazing-feature
```

3. Внесите изменения
4. Сделайте commit

```bash
git commit -m "Add amazing feature"
```

5. Отправьте изменения

```bash
git push origin feature/amazing-feature
```

6. Создайте Pull Request


## Важное требование к Pull Request

Пожалуйста, оформляйте описание PR максимально подробно:

* Что именно было изменено?
* Какую проблему это решает?
* Почему было выбрано именно такое решение?
* Если изменялся интерфейс? то приложите скриншоты
